<h1 align="center">@layerzerolabs/oapp-example</h1>

## Template repository for getting started with LayerZero using either Hardhat or Foundry in one project.

### Getting Started

#### Using Foundry

```bash
forge install
forge build
forge test
```

#### Using Hardhat

```bash
pnpm
pnpm hardhat compile
pnpm hardhat test
```
